# 🚀 TG Card CRM — Инструкция по запуску

## ⚡ Быстрый старт (фронтенд, демо-режим)

```bash
npm install
npm run dev
```
Открыть: http://localhost:5173  
Логин: **admin** / Пароль: **admin123**

> В демо-режиме всё работает с тестовыми данными. Для реального Telegram нужен Python backend.

---

## 🐍 Запуск Python Backend (для реального Telegram)

### 1. Установить Python зависимости

```bash
pip install fastapi uvicorn telethon websockets python-multipart
```

### 2. Получить Telegram API ключи

1. Откройте https://my.telegram.org
2. Войдите с вашим номером телефона
3. Перейдите в **API Development Tools**
4. Нажмите **Create new application**
5. Заполните любое название → скопируйте **App api_id** и **App api_hash**

### 3. Вставить ключи в server.py

Откройте `backend/server.py` и замените строки:
```python
API_ID = 22150711        # ← ваш api_id
API_HASH = "свой_hash"   # ← ваш api_hash
```

### 4. Запустить backend

```bash
cd backend
python server.py
```

Должно появиться:
```
==================================================
TG CRM Backend запускается...
Документация: http://localhost:8000/docs
==================================================
```

### 5. Запустить фронтенд

В другом терминале:
```bash
npm run dev
```

## 📱 Подключить Telegram аккаунт через CRM

После запуска backend:

1. Откройте CRM: http://localhost:5173
2. Перейдите в **Настройки → Аккаунты**
3. В блоке **🔑 Telegram API** вставьте ваш API ID и API Hash → нажмите **Сохранить ключи**
4. В блоке **📱 Подключить TG аккаунт**:
   - Введите номер телефона (формат: +79001234567)
   - Введите название (например: "Основной", "TikTok акк")
   - Нажмите **📨 Отправить код**
5. Откройте Telegram на телефоне — придёт код подтверждения
6. Введите код в поле → нажмите **✓ Подтвердить**
7. Если включена двухфакторная аутентификация — введите пароль → **🔑 Войти**

✅ Аккаунт подключён! Теперь в разделе **Чат** нажмите кнопку **🔄 [название акка]** чтобы загрузить все диалоги.

## 💬 Как начать обрабатывать лидов

### Шаг 1: Добавить лида
- Раздел **Лиды** → кнопка **+ Добавить**
- Укажите: имя, @username из Telegram, телефон, источник
- Выберите TG аккаунт через который общаетесь
- Нажмите **Сохранить**

### Шаг 2: Открыть чат
- В строке лида нажмите кнопку **💬**
- Или из раздела Лиды кликните на строку → в модалке нажмите **💬 В чат**

### Шаг 3: Писать сообщения
- Введите текст в поле ввода → **Enter** или кнопка **↑**
- Введите `/` для быстрых ответов из вашего списка
- Сообщение отправляется через TG аккаунт привязанный к лиду

### Шаг 4: Менять статус
- В таблице лидов кликните на цветной статус-бейдж → выберите новый
- Или откройте модалку лида → поле **Статус**
- Или в профиле чата → кнопка **✏️ Изменить**

## 🔧 Быстрые ответы (Quick Replies)

Настройте заготовленные ответы:

1. **Настройки → 💬 Быстрые ответы**
2. Нажмите **+ Добавить**
3. Укажите ярлык (например: `привет`) и текст ответа
4. В чате введите `/привет` → выберите из списка → отправьте

> **Telegram Business Quick Replies**: если у вас подключён Telegram Business и настроены готовые ответы в самом приложении Telegram — они работают на уровне Telegram и не передаются через API. Через CRM вы можете создать свои аналогичные ярлыки в разделе **Быстрые ответы**.

## ❓ Частые вопросы

**Q: Backend недоступен — это нормально?**  
A: Да, CRM работает в демо-режиме. Для реального TG нужен Python backend.

**Q: Как загрузить переписку из Telegram?**  
A: Чат → нижняя панель → кнопка **🔄 [название аккаунта]**. Загрузит последние 100 диалогов и 50 сообщений каждого.

**Q: Где хранятся данные?**  
A: В localStorage браузера. При очистке кэша данные сбросятся. Для постоянного хранения нужна база данных (доработка backend).

**Q: Как добавить второй аккаунт Telegram?**  
A: Настройки → Аккаунты → повторите шаги подключения с новым номером.

**Q: Можно ли видеть диалоги сразу с 2 аккаунтов?**  
A: Да! В разделе Чат в верхней части есть вкладки: **Все | Основной акк | TikTok акк**. Переключайтесь или смотрите все сразу.
